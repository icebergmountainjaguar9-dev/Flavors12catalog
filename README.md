# Flavors12catalog

This repository contains a static landing page for `flavors12catalog.com`.

## Files

- `index.html` — main page with flavor cards, emoji illustrations, and order CTA.
- `styles.css` — responsive styling for the site.

## Deploying to Vercel

1. Install the Vercel CLI or use the Vercel dashboard.
2. Connect this repository or deploy from the root directory.
3. Vercel will automatically serve `index.html` as a static site.

The site is ready to deploy and works well on Vercel as a static frontend.

## Crypto Checkout

The page now includes Bitcoin and Ethereum checkout sections. It polls public blockchain APIs to detect payments to the configured addresses and updates the confirmation status automatically.
